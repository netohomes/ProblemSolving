# 416. Partition Equal Subset Sum

## Problem definition

https://leetcode.com/problems/partition-equal-subset-sum/description/

## DP solution

Below is the solution to find if the target sum can be gotten. Basically we marks all the possible results that can be achieved by adding the numbers.

```c++
bool canPartition(vector<int>& nums) {
    int sum = accumulate(nums.begin(), nums.end(), 0);
    int target = sum/2;
    if (sum%2) {
        return false;
    }
    vector<bool> dp = vector<bool>(target+1, false);
    dp[0] = true;
    for (int num: nums) {
        for (int i = target; i >= num; i--) {
            dp[i] = dp[i] || dp[i-num];
            if (dp[target]) {
                return true;
            }
        }
    }
    return dp[target];
}
```
